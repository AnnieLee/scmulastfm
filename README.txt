Para a execução da aplicação é preciso fazer import de algumas bibliotecas.
Todas as bibliotecas necessárias estão guardadas na pasta libs, pelo que só é preciso importá-las depois para o projecto: Project -> Properties -> Java Build Path -> Libraries -> Add JARs... e seleccionar as bibliotecas que se encontram na pasta libs do projecto MobileLastFM

Para além disso é preciso correr o projecto paralelamente com os serviços da google, este projecto por sua vez é preciso incorporar com a aplicação como uma biblioteca. Project -> Properties -> Android -> Library -> Add… e seleccionar google-play-services_lib

Este projecto é enviado em conjunto com o código da aplicação.

Para além disso o professor terá que inserir a chave da API do Google no AndroidManifest.xml.
